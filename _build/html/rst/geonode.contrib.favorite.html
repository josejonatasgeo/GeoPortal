geonode.contrib.favorite.utils.get_favorite_info(user, content_object)¶
return favorite info dict containing:
  1. an add favorite url for the input parameters.
  2. whether there is an existing Favorite for the input parameters.
  3. a delete url (if there is an existing Favorite).

geonode.contrib.favorite.views module¶

geonode.contrib.favorite.views.delete_favorite(request, *args, **kwargs)¶

delete favorite and put favorite_info object in response. method is idempotent, if favorite does not exist, just return favorite_info.

geonode.contrib.favorite.views.favorite(request, *args, **kwargs)¶

create favorite and put favorite_info object in response. method is idempotent, Favorite’s create_favorite method only creates if does not already exist.
