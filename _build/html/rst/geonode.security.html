set_permissions(perm_spec)¶

Sets an object’s the permission levels based on the perm_spec JSON.

the mapping looks like: {

‘users’: {
‘AnonymousUser’: [‘view’], <username>: [‘perm1’,’perm2’,’perm3’], <username2>: [‘perm1’,’perm2’,’perm3’] …

} ‘groups’: [

<groupname>: [‘perm1’,’perm2’,’perm3’], <groupname2>: [‘perm1’,’perm2’,’perm3’], … ]

}

geonode.security.utils.set_geofence_owner(*args, **kwargs)¶

Assign access permissions to owner user

This function uses the geoserver REST API in order to contact geofence. The request performs a similar function to the one in the following cURL snippet:

curl -X POST -u {admin_user}:{admin_password} -H “Content-Type: text/xml” -d “<Rule><userName>{user}</userName><workspace>geonode</workspace> <layer>{layer}</layer><access>ALLOW</access></Rule>” http://<host>:<port>/geoserver/geofence/rest/rules
